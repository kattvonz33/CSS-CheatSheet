CSS Selectors & Specificity



Selectors

Name Description CSS Example HTML Example
Type Matches the type of the element in the HTML doc p {}, div{} <p> , <div>
Universal Selects all elements of any type * {}
Class One of the most common ways to select an element ".class" <p class='class'>
ID Used for uniquely styling an HTML element
(IDs override types and classes) 		#description <h1 id='description'>
Attribute Used to target HTML elements that already contain attributes img[src*='winter']{height: 50px} <img src='/images/winter.jpg'>
Pseudo-Class Changes the appearance/state of certain elements, after certain interactions: Focus, Visited, Disabled, Active, Hover p:hover {background-color:blue}
